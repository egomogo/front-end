### [이거머거] frond-end

### ✉️ Commit Type
▫ Feat : 새로운 기능 추가, 기존의 기능을 요구 사항에 맞추어 수정 <br/>
▫ Fix : 기능에 대한 버그 수정 <br/>
▫ Build : 빌드 관련 수정 <br/>
▫ Chore : 패키지 매니저 수정, 그 외 기타 수정 <br/>
▫ Docs : 문서(주석) 수정 <br/>
▫ Style : 코드 스타일, 포맷팅에 대한 수정 <br/>
▫ Refacto : 기능의 변화가 아닌 코드 리팩터링 <br/>
▫ Test : 테스트 코드 추가/수정 <br/>
▫ Release : 버전 릴리즈 <br/>
▫ ex) [Refacto] 변수 이름 변경 <br/>
